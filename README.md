🏦 Bank Management System (Python Project)
📌 Project Description

The Bank Management System is a Python-based application developed to manage basic banking operations digitally. This system allows users to create accounts, deposit and withdraw money, and check account balances efficiently. It replaces manual record-keeping with an automated system, improving accuracy and speed.

🎯 Objectives
To automate banking operations
To reduce human errors
To maintain secure and organized records
To provide quick access to account information
⚙️ Features
Create new bank accounts
Deposit money into accounts
Withdraw money from accounts
Check account balance
Simple and user-friendly interface (CLI-based)
🛠️ Technologies Used
Python (Core Programming)
CLI (Command Line Interface)

▶️ How to Run
